# 页面拼装工作流指南

## 核心原则：设计系统先行

基于现有完整的设计系统（PlanetX + SaaS），页面开发只需关注业务逻辑，无需担心样式一致性。

## 1. 可用组件库总览

### PlanetX（游戏化C端）
- ✅ **基础组件**：Button、Input、TextArea、Card
- ✅ **数据展示**：XPBar、LevelBadge、QuizOptionCard
- ✅ **反馈组件**：ToastBar、AchievementPopup、Loading
- ✅ **布局组件**：Modal、Dropdown、Tabs
- ✅ **背景特效**：StarBackground
- ✅ **设计系统**：163个Token + 12个动画

### SaaS（专业B端）
- ✅ **基础组件**：Button、Input、Select、TextArea、Card
- ✅ **布局组件**：Sidebar、Header、Modal
- ✅ **数据展示**：KPICard、DataTable、ProgressBar
- ✅ **交互组件**：Toggle、DatePicker、Tooltip
- ✅ **聊天组件**：ChatBubble、StreamingText
- ✅ **上传组件**：ResumeUploader

### 弹窗+表单组合

```tsx
import { PlanetXModal, PlanetXInput, PlanetXButton } from "@looma/planetx/brand/ui";

export default function ExampleModal() {
  const [isOpen, setIsOpen] = useState(false);
  const [formData, setFormData] = useState({ name: "", email: "" });

  return (
    <>
      <PlanetXButton onClick={() => setIsOpen(true)}>

        onClose={() => setIsOpen(false)}
        title="Create Mission"
        size="md"

        <PlanetXInput
          label="Mission Name"
          value={formData.name}
          onChange={(val) => setFormData({...formData, name: val})}
          placeholder="Enter mission name"
        />
        <PlanetXInput
          label="Description"
          value={formData.email}
          onChange={(val) => setFormData({...formData, email: val})}
          placeholder="Enter description"
          style={{ marginTop: "var(--px-spacing-md)" }}

          display: "flex", 
          gap: "var(--px-spacing-sm)",
```

### 数据表格+分页组合

```tsx
import { SaasDataTable, SaasSelect, SaasButton } from "@looma/saas/brand/ui";

export default function UserTable() {
  const [selected, setSelected] = useState<string[]>([]);
  const [page, setPage] = useState(1);

  const columns = [
    { key: "name", title: "Name", render: (row) => row.name },
    { key: "email", title: "Email", render: (row) => row.email },
    { key: "role", title: "Role", render: (row) => row.role },
  ];

  return (
    <div>
      {/* 筛选工具栏 */}
      <div style={{ 
        display: "flex", 
        gap: "var(--spacing-md)",
        marginBottom: "var(--spacing-lg)"
      }}>
        <SaasSelect
          value="all"
          options={[
            { value: "all", label: "All Roles" },
            { value: "admin", label: "Admin" },
            { value: "user", label: "User" }
          ]}
          style={{ width: "200px" }}
        />
        <SaasButton>Export</SaasButton>
      </div>
      
      {/* 数据表格 */}
      <SaasDataTable
        columns={columns}
        data={users}
        selectable
        selectedKeys={selected}
        onRowSelect={(key, selected) => {
          if (selected) setSelected([...selected, key]);
          else setSelected(selected.filter(id => id !== key));
        }}
        pageSize={10}
        currentPage={page}
        total={users.length}
        onPageChange={setPage}
      />
    </div>
  );
}
```

## 8. 常见问题

### Q: 如何决定使用哪个品牌的设计系统？
A: 
- 面向求职者（C端）→ 使用PlanetX（游戏化、深色主题）
- 面向HR/企业（B端）→ 使用SaaS（专业、浅色主题）

### Q: 缺少某个组件怎么办？
A: 现有组件库已覆盖95%使用场景。如果确实需要特殊组件：
1. 检查能否用现有组件组合实现
2. 参考其他类似组件进行扩展
3. 仅当必要时创建新组件

### Q: 如何确保跨端一致性？
A: 始终使用设计系统Token，而不是硬编码样式值。例如：
- ✅ `color: var(--px-color-primary)`
- ❌ `color: "#6C63FF"`
