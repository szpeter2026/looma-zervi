### 1.3 PlanetX-T空间 TAM / SAM / SOM 估算

| 层级 | 定义 | 估算 | 计算逻辑 |
|------|------|------|----------|
| **TAM**（总可寻址市场） | 中国求职招聘+心理测评+AI招聘 SaaS 总和 | ~¥650 亿 | ¥450亿(网络招聘) + ¥80亿(心理测评) + ¥120亿(AI招聘SaaS) |
| **SAM**（可服务市场） | 面向 GEN-Z 求职者 + 中小企业的 AI 驱动细分 | ~¥80 亿 | TAM × 12%（GEN-Z+中小企业渗透率） |
| **SOM**（可获得市场） | 通过微信小程序+裂变传播可触达的份额（3 年目标） | ~¥2.4 亿 | SAM × 3%（保守估计，依赖裂变+免费策略快速渗透） |

> **备注：** SOM 按 3 年目标估算，对应日活 5 万求职者 + 3000 家付费企业。若裂变效果超预期（对标 MBTI 小程序的传播速度），SOM 可上修至 ¥5-8 亿。

### 1.4 用户获取成本（CAC）估算

| 渠道 | 预估 CAC | 说明 |
|------|---------|------|
| 自然裂变（3人成团/星球名片分享） | **¥0-2** | 核心获客引擎，MBTI 人格测试结果天然具备社交传播属性 |
| 微信搜一搜 SEO | ¥3-5 | "AI 求职""模拟面试""MBTI 职业匹配"等长尾关键词 |
| 小红书/抖音内容种草 | ¥8-15 | GEN-Z 求职焦虑 + MBTI 人格话题天然适合内容营销 |
| 校园推广大使 | ¥5-10 | 与高校就业指导中心合作，每校 2-3 名校园大使 |
| 付费投放（备选） | ¥20-40 | 仅在企业端规模化后考虑，C 端以免费裂变为主 |

> **对标参考：** Boss直聘 CAC 约 ¥50-80（含品牌广告），求职精灵预估 CAC ¥15-30（信息流投放）。PlanetX 的 CAC 目标显著低于行业均值，核心依赖**社交裂变引擎**。

### 1.5 各赛道市场集中度（CR3）

| 赛道 | CR3 估算 | 竞争格局 |
|------|---------|----------|
| AI 求职工具 | ~45% | 分散，求职精灵/超级简历/五百丁各有细分，无绝对龙头 |
| 简历工具 | ~55% | 中等集中，全民简历+五百丁+Wondercv 三家领先 |
| MBTI 心理测评 | ~30% | 高度分散，小程序生态数百款同质产品，CR3 极低 |
| AI 招聘 SaaS | ~75% | 高度集中，Moka+北森+大易占据中大型企业市场 |
| 情绪陪伴 | ~40% | 中等分散，潮汐+小睡眠+Know Yourself 各有侧重 |

> **关键判断：** MBTI 心理测评赛道高度分散（CR3 仅 30%），意味着**无需对抗巨头即可快速获得市场份额**。AI 招聘 SaaS 赛道高度集中，但 T空间 瞄准的是 CR3 未覆盖的**中小企业长尾市场**。

### 1.6 赛道交叉的独特性

**关键洞察：** 目前市场上**没有任何一个产品**同时覆盖 A+B+C+D+E 五个赛道。PlanetX-T空间 的核心机会在于——以"AI求职"为入口，以"心理测评+情绪陪伴"为留存钩子，以"企业招聘SaaS"为商业变现，以"诗词/文化内容"为品牌调性，构建一个**求职者的全生命周期陪伴生态**。

## 二、竞品分析：求职精灵

### 2.1 基本信息

| 维度 | 详情 |
|------|------|
| **产品全称** | 求职精灵（Job Seeking Genie / 求职精灵） |
| **所属公司** | 推测为独立创业团队或某招聘平台孵化项目 |
| **产品形态** | 微信小程序 + H5 |
| **核心定位** | AI 驱动的求职助手：简历优化、职位推荐、模拟面试 |
| **目标用户** | 应届生 + 工作 0-5 年初入职场人群 |
| **商业模式** | 基础免费 + 会员订阅（¥19.9-¥49.9/月） |
| **上线状态** | 已上线运营 |

> **信息获取方式：** 公开资料 + 微信小程序搜索 + 部分实测体验  
> **备注：** "求职精灵"为通用名称，市场上可能存在多个同名或类似产品。以下分析基于该品类代表性产品的共性特征。标注「推测」的部分为基于公开信息的合理推断，需后续实测验证。

### 2.2 产品功能拆解

| 模块 | 功能点 | PlanetX 对标状态 |
|------|--------|-----------------|
| **简历优化** | AI 简历评分、关键词补充、格式美化、多版本生成 | ⚠️ 已有简历解析（`/v1/resume/parse`），缺少主动优化建议 |
| **职位匹配** | 基于简历自动推荐职位、匹配度打分 | ✅ 已有 `/v1/jobs/match` |
| **模拟面试** | AI 面试官提问、语音/文字回答、评分报告 | ❌ 缺失（PlanetX-T空间 规划中 P0） |
| **求职课堂** | 面经、笔试题库、行业科普 | ❌ 缺失（可规划 P2） |
| **投递追踪** | 投递记录、面试日程、offer 管理 | ❌ 缺失（PlanetX 规划中 P1） |
| **个性化推荐** | 基于用户行为+简历的智能推荐算法 | ⚠️ 部分已有（匹配基于文本相似度，缺少行为数据） |

### 2.3 优势分析

| 优势 | 说明 |
|------|------|
| **垂直聚焦** | 100% 聚焦求职场景，功能深度优于通用平台 |
| **上手简单** | 小程序形态，即开即用，无需下载 |
| **AI 成熟度** | 简历优化+面试模拟有成熟的 NLP 方案 |
| **用户认知** | "求职助手"概念已被市场教育 |

### 2.4 劣势与短板

| 劣势 | PlanetX 可攻击点 |
|------|-----------------|
| **单点工具，无生态** | 求职结束后用户必然流失。PlanetX 有情绪陪伴+组队+成长路径，可延长 LTV |
| **无企业端** | 只有 C 端，无法形成供需闭环。PlanetX-T空间 有 B 端 T空间 |
| **品牌调性弱** | 工具属性强，缺乏情感连接。PlanetX 的"星球""重装登陆"有强 IP 潜力 |
| **无社交裂变** | 纯工具，用户不会自发传播。PlanetX 的"3人成团解锁星图"有天然裂变机制 |
| **无心理测评** | 缺少人格匹配维度。PlanetX 的星际人格测试可提供"性格→职业"匹配 |

### 2.5 与 PlanetX 的差异化策略

```
求职精灵：AI 帮你找到工作  →  工具属性，用完即走
    vs
PlanetX：AI 陪你成为更好的自己  →  陪伴属性，长期粘性

核心差异化打法：
1. 求职精灵做"简历→职位"单向匹配
   PlanetX 做"人格+技能+情绪"三维匹配（评测→匹配→陪伴→成长）
2. 求职精灵无社交
   PlanetX 有"3人成团"组队建舱社交裂变
3. 求职精灵求职结束=用户流失
   PlanetX 求职结束=进入成长路径+情绪陪伴，用户长期留存
```

### 3.3 优势分析

| 优势 | 说明 |

| **品牌老化** | "全民简历"调性偏传统，对 00 后/GEN-Z 吸引力弱 |

## 四、竞品分析：MBTI 心理测评赛道

### 4.1 赛道概述

MBTI（Myers-Briggs Type Indicator）在中国 GEN-Z 群体中已成为现象级社交货币。从"你的 MBTI 是什么"成为年轻人破冰话题，到各平台 MBTI 相关内容播放量数百亿次，MBTI 已经从心理学工具进化为**社交身份标签**和**消费决策依据**。

| 指标 | 数据 |
|------|------|
| 微博 #MBTI# 话题阅读量 | 50 亿+ |
| 小红书 MBTI 相关笔记 | 1000 万+ |
| 抖音 MBTI 相关视频播放 | 300 亿+ |
| B站 MBTI 二创视频 | 日均新增 500+ |

### 4.2 主流 MBTI 产品竞品分析

#### 4.2.1 才储（CaiChu）

> **信息获取方式：** Web 端实测 + 公开资料

| 维度 | 详情 |
|------|------|
| **产品形态** | Web + 微信公众号 |
| **核心定位** | 专业心理测评平台，以 MBTI 为核心延伸多元人格测评 |
| **测评类型** | MBTI 标准版（93题）、MBTI 专业版（200题+）、大五人格、霍兰德职业兴趣、DISC |
| **职业结合度** | ⭐⭐⭐⭐ 有 MBTI→职业推荐，但推荐逻辑较简单（静态映射表） |
| **商业模式** | 基础免费 + 深度报告付费（¥9.9-¥49.9） |
| **用户群体** | 职场人士、HR 从业者、心理学爱好者 |
| **优势** | 测评专业性公认、题目池深厚、国内 MBTI 品类开创者 |
| **劣势** | UI 老旧、无社交功能、无情绪陪伴、移动端体验差 |

#### 4.2.2 MBTI 性格测试（微信小程序）

| **测评类型** | MBTI 简版（28-60题）、九型人格 |
| **职业结合度** | ⭐⭐ 仅提供泛化的人格描述，无具体职业推荐 |
| **商业模式** | 广告变现 + 解锁报告付费 |
| **用户群体** | 年轻女性为主，社交传播驱动 |
| **优势** | 传播力强、裂变快、题目简短 |
| **劣势** | 专业性差、无职业结合、无后续服务、同质化严重 |

### 8.2 隐私保护：被忽视的竞争维度

#### 8.2.1 求职场景的隐私痛点

| 风险类型 | 具体表现 | 用户痛点 |

| **在职风险** | 在职求职者的简历可能被现雇主发现 | "HR 在 Boss直聘上刷到我的简历" |

| **AI 训练滥用** | 用户简历/面试录音被平台用于训练 AI 模型 | 用户无知情权，数据被二次利用 |

> **背景：** 《中华人民共和国个人信息保护法》（2021.11.01 施行）对求职数据属于"敏感个人信息"，要求单独同意+最小必要原则。2025 年已有多家招聘平台因数据违规被约谈/处罚。

#### 8.2.3 隐私保护的差异化价值

```
为什么隐私 = 竞争武器？

1. 用户痛点真实且强烈：
   GEN-Z 对隐私的敏感度远超上一代，"数据主权"是他们的核心诉求
   → "我的简历凭什么被你拿去训练 AI？"

2. 竞品集体沉默：
   没有任何一家竞品把隐私保护作为差异化卖点
   → 这是一个完全无人占领的品牌心智高地

3. 合规壁垒：
   《个人信息保护法》日趋严格，先建立隐私保护体系的产品将拥有合规先发优势
   → 当竞品被迫整改时，PlanetX 已完成品牌信任积累

4. 技术可实现：
   looma-zervi 后端已有「解析不落盘」「用完即焚」的架构基础
   → 不是空头承诺，而是可验证的技术能力
```
